How to make a request?

To make a request, you have to follow these steps:

  1. Choose one of the endpoints
  2. You'll need to add the following query strings to every request:
    • Version
    • DeveloperID
  3. You'll need to add the following fields to the header of your request:
    • hmac
  4. You're now ready to make the request
What is value for Version?

It can be any arbitary value. For example Version=1.0

What is a DeveloperID and how do I get it?

A DeveloperID is used to identify you from the request. You will get one in an email when you signup.

What is HMAC and how do I generate one?

HMAC a digital signature that helps us make sure that you have made the request. You can generate one by encrypting the request using your Private Key. Use the SHA256 algorithm. You must generate one for each request.

For example, call hash_hmac("sha256", $request, $privatekey) in PHP.

An example request to get the data for an album: 
http://www.bollywoodapi.com/v1/albums/s_123?DeveloperID=456&Version=1.0

Include the the following header field

hmac: 2347324y7jsdfasdjn234
