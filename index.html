How to make a request?

To make a request, you have to follow these steps:

  1. Choose one of the endpoints
  2. You'll need to add the following query strings to every request:
    • DeveloperID
    • Timestamp
    • hmac
  3. You're now ready to make the request
What is a DeveloperID and how do I get it?

A DeveloperID is used to identify you from the request. You will get one in an email when you signup.

What is a Timestamp?

A unix timestamp is the number of seconds since January 1, 1970. Most languages have an in-built function. For example, call time() in PHP.

What is HMAC and how do I generate one?

HMAC a digital signature that helps us make sure that you have made the request. You can generate one by encrypting the Timestamp using your Private Key. Use the SHA256 algorithm. You must generate one for each request.

For example, call hash_hmac("sha256", $timestamp, $privatekey) in PHP. Make sure that the timestamp is the same as the one that you are sending as a query string.

An example request to get the data for an album: 
albums/s_123?DeveloperID=456&Timestamp=1389195200&hmac=7d1fce595ce8f9a0a724c76ecba700420fca0031fd9dca1fdf3
